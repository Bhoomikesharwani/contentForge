# ContentForge 🚀

ContentForge is a modern AI-powered content generation platform designed to help users create high-quality digital content quickly and efficiently. The platform provides features such as AI content generation, authentication, smart workflows, and user-friendly content management tools.

## ✨ Features

* 🔐 Secure Authentication
* 🤖 AI-Powered Content Generation
* 📄 Dynamic Content Management
* ⚡ Fast and Responsive UI
* 🌐 Modern Full-Stack Architecture
* 📱 Fully Responsive Design

## 🛠️ Tech Stack

### Frontend

* Next.js
* React.js
* Tailwind CSS

### Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication

## 📂 Project Structure

```bash
frontend/
backend/
```

## 🚀 Installation

### Clone Repository

```

### Install Frontend

```bash
cd frontend
npm install
npm run dev
```

### Install Backend

```bash
cd backend
npm install
npm start
```

## 🔑 Environment Variables

Create `.env` files in frontend and backend folders.

### Backend `.env`

```env
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
```
